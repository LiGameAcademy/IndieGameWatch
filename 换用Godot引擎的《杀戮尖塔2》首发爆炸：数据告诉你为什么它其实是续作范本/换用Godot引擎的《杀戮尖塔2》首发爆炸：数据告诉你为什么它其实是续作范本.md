# 换用Godot引擎的《杀戮尖塔2》首发爆炸： 

**核心论点：** 《杀戮尖塔2》的“没诚意”，恰恰是独立游戏界最顶级的“克制”与“清醒”。

## 一、 开场：从“像个 DLC”到“续作范本”

先说结论：**我一开始也觉得《杀戮尖塔2》有点“没诚意”。**

刚开抢先体验那几天，我的第一反应可能和很多人一样：  
“这不就是一代换了套皮、加了俩职业、调了调数值吗？怎么就二代了？”  
甚至在动态里半开玩笑地说过一句：**“这更像是一个超大号 DLC。”**

然后评论区直接炸锅：  
有人认同，觉得二代确实不够激进；  
也有人很不爽，觉得这种评价是在否定制作组这几年的打磨。

这期视频，其实就是我对那句“像 DLC”的一次**认错 + 复盘**。  
过去这几天，我把首发数据、SteamDB 的在线人数、媒体报道、还有几千条中文评论都翻了一遍，  
然后再从**开发者的视角**，重新去看这代游戏做了什么、没做什么。

看完之后，我现在更愿意这样形容《杀戮尖塔2》：  
> **“它不是那种会颠覆一切的神作续作，  
> 但它是独立游戏圈里，极其罕见的一种‘清醒’。”**

它证明了一件看似普通，却极难做到的事情：  
**MegaCrit 非常清楚一代到底为什么成功，也非常清楚自己不该乱改什么。**  
在巨大的商业压力、玩家期待、以及“做点不一样的东西”的冲动之下，  
他们选择了**克制**——在保留原配方的前提下，把整套东西用 Godot 重做了一遍。

这期我们就从三个角度聊聊这件事：  
**玩家视角的数据、设计视角的取舍、以及开发者视角下的 Godot 选择。**

## 二、数据说话：为什么说它依然是成功的？

我们先把情绪放一边，**先看几组很冷冰冰的数据**。

我这次是分了两路去查：

- 一路是 SteamDB、媒体报道这种“公开大盘数据”；
- 一路是我自己写了个小脚本，从 Steam API 抓了几千条**简体中文评论**下来，专门看国区玩家到底怎么玩、怎么评价。

先看大盘。  
根据 SteamDB 和几家媒体的统计：

- **《杀戮尖塔 1》在 Steam 上的历史峰值在线人数，大概是 5.7 万人左右。**
- 而 **《杀戮尖塔 2》抢先体验刚开的时候，在线人数一路飙到四十多万、最高接近五十多万，**  
  是一代历史峰值的**将近 10 倍**，一度把 Steam 挤到半瘫痪。
- 在 Twitch 上，二代首发当天的**观看峰值也在 13 万人左右**，属于全平台前排的热度。

光看这三行数字，其实已经可以得出一个很现实的结论：  
**“保持原配方”的《杀戮尖塔 2》，在商业上是极其成功的。**  
它不仅稳稳接住了一代的老玩家，还拉来了一大批新观众、新主播，把整个品类的天花板又往上抬了一截。

但是，单看“多少人买、多少人在看”还不够。  
一个肉鸽卡牌值不值，它真正的战场在于——**大家到底玩了多久。**

所以我又从 Steam 的评论接口里，抓了 3000 多条简体中文的评测，专门看这批玩家的游玩时长。  

在这 3200 条中文评论里：

- 玩家**平均游玩时长是 26.2 小时**；
- **中位数游玩时长是 24 小时**——不是被几个肝帝拉高的“虚高平均数”，而是大多数人真真切切玩了二十多个小时；
- 其中，**有 61.03% 的玩家游玩时长超过了 20 小时。**

你回想一下我们刚才的讨论：  
在单机肉鸽卡牌这个品类里，我个人给自己的一个“性价比心理线”是——**买完能玩 20 小时以上，这笔钱就算值。**  
从这个角度看，《杀戮尖塔 2》在 EA 初期就已经把**大多数玩家送到了“值回票价线”之上**，而且后面还会有大版本更新、Mod 生态、平衡迭代，这些时长只会继续往上长。

所以，当有人说“二代像 DLC”、“内容不够多”的时候，我的感觉是这样的：

- **从“创新幅度”的角度，这个吐槽是有道理的；**
- 但从“你花的钱换来了多少小时的体验”这个角度，**数据是站在游戏这边的。**

你可以觉得它“不够惊喜”，  
你可以失望它“没有彻底颠覆自己”，  
但要说它“不值这个价”，  
**在这堆实打实的时长统计和 CCU 曲线面前，其实有点站不住脚。**

而这，恰恰是我在复盘完数据之后，开始重新审视这款游戏的第一个转折点——  
**它也许真的不是一款“伟大的 2 代”，  
但它是一款异常清醒、异常务实的“续作范本”。**

## 三、 深度解析：为什么“再做一遍”没什么不对？

很多人对续作有一个天然的期待：**“必须不一样。”**  
但问题在于，游戏设计不是写论文，创新本身不是目标，**好玩才是。**

如果你把一代当成一个“齿轮组”，那《杀戮尖塔》的设计原点，大概可以浓缩成三件事：

- **能量 / 费用系统**：让你每回合都在做痛苦选择。
- **过牌与抽滤**：让你不断在“赌未来的自己”。
- **遗物 + 卡牌的化学反应**：让每一把 run 都长得完全不一样。

这套齿轮组在一代的时候，其实已经**精密到有点危险**了：  
任何一个部件改得太大，整个系统都可能散架。  

我们已经见过太多“想要大改”的续作翻车案例了。  
最典型的就是《暗黑地牢2》：  
第二代在战斗结构、世界结构上做了很多激进的调整，结果就是——  
对一代老玩家来说，它**像是同一套美术下的另一款游戏**，而不是“更好的暗黑地牢”。

回头看《杀戮尖塔2》，你会发现它做的事情反而“朴素”得多：

- **不改动齿轮的核心结构**：费用、手牌、遗物的关系基本保持不变。
- **在齿轮上“抛光”和“扩展”**：新角色、新机制、新怪物，都是在旧规则上叠加，而不是推倒重来。
- **把更多精力花在“表达”和“可玩性密度”上**：剧情演出更完整，世界观更统一，鸡煲和新卡组玩起来的“上头感”是延续而不是取代。

站在玩家心理的角度，这样的选择其实非常现实：  
**核心粉丝买二代，很多时候不是为了“换游戏”，而是为了“多一台更大的同款老虎机”。**  
他们想要的是：

- 熟悉的那种“越打越懂”的决策压力；
- 但是配合全新的卡牌、遗物、流派组合；
- 最好还能多一点剧情、多一点细节、多一点“新东西可以研究”。

从这个角度看，《杀戮尖塔2》的问题不是“再做了一遍”，  
而是**很多宣传和玩家预期，把它当成了一个“必须要颠覆自我”的 2 代。**  

如果你把预期调成“**1.5 代内容量 + 2 代级别的技术和美术翻新**”，  
那这款游戏现在交出的答卷，其实是相当稳的。

## 四、 开发者视角：为什么“再做一遍”其实非常难？（顺便聊聊 Godot）

前面我们更多站在**玩家**的角度，去看这款游戏值不值。  
接下来我们换一个视角：**如果你是开发者，你会发现 MegaCrit 干了一件非常冒险的事。**

外行看二代，很容易得出一个结论：  
“这不就是一代那套玩法，再换一遍美术、顺便多写点剧情吗？”  
但实际上，《杀戮尖塔2》在背后做了一个非常大的决定：——**底层引擎直接从原来的旧技术栈，迁移到了 Godot。**

对一个已经成功的项目来说，换引擎意味着什么？

- **几乎所有底层代码都要重写**：包括战斗循环、状态机、动画、存档系统、事件系统等等；
- **所有微妙的手感都要重新找一遍**：出牌的响应、缓动、音效延迟、特效时机，这些在旧引擎里调好了，在新引擎里得全部重来；
- **所有复杂的结算逻辑都要重新验证**：一代那么多 Buff、Debuff、遗物组合，要在新引擎里重新保证“不炸服”，其实是非常恐怖的工作量。

换句话说，**MegaCrit 一边在做“再来一次”，一边其实是在“从零重构整个项目的底层架构”。**  

如果你自己做过一点项目，你应该会有这样的经验：  
**做一个小 demo 很容易；**  
真正在同一个项目里，撑住几百张卡牌、上百个遗物、各种状态和事件互相咬合，  
如果架构没设计好，后期就会变成——

- 改一个 bug，生出十个新 bug；
- 每次加新卡，都要在一堆 if-else 迷宫里打补丁；
- 越往后越没人敢动核心逻辑，因为一动就炸存档。

我这次之所以会用《杀戮尖塔2》作为例子，  
就是想告诉大家：**你屏幕上看到的那种“没什么变化的二代”，背后其实是一次完整的架构升级。**  

这也是为什么，这两年我在 B 站和课堂里，会一直做各种 **Godot 4 架构实战**：  

- 用 Godot 做一个小 demo 很快，随便拖两个节点就能跑起来；
- 但要做出一款可以持续更新、支持复杂战斗和系统联动的商业项目，**拼的已经不是 API 熟不熟，而是底层架构有没有想清楚。**

无论是我在课程里讲的回合制战斗、技能系统，还是对话、关卡编辑器，本质上都是在教一件事：  
**怎样在 Godot 里，搭出一套“能承受你未来野心”的项目骨架。**  

如果你看完这期，对《杀戮尖塔2》的底层有一点点好奇，  
欢迎来我的 Godot 4 实战课程里，一起拆解“怎么用一个开源引擎，撑起类似体量的系统”。

## 五、 总结与引导 (7:30 - 结尾)

如果把我们刚才说的这些拉远一点，会得到一个还挺朴素的结论：  

- 从 **商业数据** 来看，《杀戮尖塔2》是一次非常成功的续作；
- 从 **玩家游玩时长** 来看，它对得起现在这个定价；
- 从 **设计与引擎选择** 来看，它是一份非常清醒、非常务实的答卷。

你可以失望它没有“推翻一切重来”；  
你可以吐槽它在画面、UI、创新上的遗憾——从我们的评论数据里看，确实有一小撮人和我一样在意这些。  
但如果要问一句：**这款游戏值不值得、MegaCrit 做得是不是对的？**  
我的答案会是：在当下这个独立游戏环境里，**这是一个非常聪明、也非常难得的选择。**

所以，别再只盯着那几张截图吐槽画质了。  
能在换了 Godot 引擎的前提下，稳稳当当地把一款复杂的卡牌游戏“再做一遍”，
还把一代的成绩放大到十倍，这本身就是实力的证明。
